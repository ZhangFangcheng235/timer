# 计时器应用

一个功能丰富的在线计时器应用，支持多种计时模式和自定义设置。

## 功能特点

- **倒计时模式**：设置特定时间进行倒计时
- **正计时模式**：从零开始计时
- **番茄工作法**：25分钟工作时间，5分钟休息时间的循环
- **自定义计时器**：创建多个自定义计时器
- **声音提醒**：计时结束时播放提示音
- **进度条显示**：直观显示计时进度
- **响应式设计**：适配各种屏幕尺寸

## 使用方法

1. 打开index.html文件
2. 选择计时模式（倒计时、正计时或番茄工作法）
3. 设置时间参数
4. 点击开始按钮启动计时器
5. 可随时暂停、继续或重置计时器

## 自定义选项

- 可自定义工作时长和休息时长
- 可选择不同的提示音
- 可调整计时器显示样式
- 支持键盘快捷键操作
